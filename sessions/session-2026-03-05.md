Commands like

PARTITION BY

ROWS BETWEEN

Examples like  
Running average of weights ordered by brick id
SELECT brick_id, weight,
round(avg(weight) over (order by brick_id), 2)
Counting bricks per shape using window functions
SELECT b.,
count() over (partition by shape)
Finding values using window frames
SELECT b.*,
min(colour) over (order by brick_id rows between 2 preceding and 1 preceding)
